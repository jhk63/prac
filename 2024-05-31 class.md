### 복사 생성자

+ 자신과 같은 클래스 타입의 다른 객체에 대한 참조(reference)를 인수로 전달받아, 그 참조를 가지고 자신을 초기화하는 방법이다.

```cpp
class A
{
    A(A& a)
    // or
    A(const A& a)
}
```

+ 무브시멘틱(Move Semantic) - cpp에서 제공

+ abstract class에서 생성자 함수를 호출할 수 없다. 
+ private로 생성자 함수를 숨길 수 있다.
